## Description

Les impacts environnementaux ne se limitent pas à la simple consommation d'électricité des serveurs d'IA.

Pour être exhaustif, il convient également d'intégrer la performance environnementale sur tous les indicateurs du centre
de données qui les met à disposition.

On peut citer notamment :
* Le PUE : Le « Power Usage Effectiveness » (Efficacité de l'utilisation de l'énergie) est un indicateur de performance
utilisé pour mesurer l'efficacité énergétique des centres de données. Il est calculé en divisant l'énergie totale
consommée par le centre de données (systèmes auxiliaires, systèmes de refroidissement...) par l'énergie utilisée par
les équipements informatiques (serveurs, stockage, réseau).

* WUE : Le « Water Usage Effectiveness » (Efficacité de l'utilisation de l'eau) est un indicateur de performance utilisé
pour mesurer l'efficacité en eau des centres de données. Il est calculé en divisant la quantité d'eau utilisée par le
centre de données par l'énergie utilisée par les équipements informatiques (serveurs, stockage, réseau).


## Exemple

* PUE : Plus le PUE il approchera 1,0, plus le centre de données propose une plus grande efficacité énergétique.
En 2023, le PUE moyen est de 1,58.
* WUE : En 2023, le WUE moyen aux États-Unis est de 1,8L/kWh. Une bonne performance sera en dessous de 0,5L/kWh.
