# Référentiels de bonnes pratiques pour l'intégration d'IA (RIA)

Ce projet regroupe des bonnes pratiques pour l'IA générative dans les services numériques.

Ce document est issu du travail des consultants du consortium DECENZ constitué des entreprises (par ordre alphabétique) :

- Constellations
- Davidson Consulting
- Ekimetrics
- Norsys
- Zenika
