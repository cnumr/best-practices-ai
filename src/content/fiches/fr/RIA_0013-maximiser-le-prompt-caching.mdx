## Description

Les fournisseurs d’outils d’IA générative ont mis en place des mécanismes d’optimisations comme le prompt caching : les
fragments de prompts répétés et placés de façon similaires sont mis en cache de leur côté.

L’utilisation de ce mécanisme permet donc de réduire les ressources utilisées.
Voir par exemple la documentation d'Open AI : https://platform.openai.com/docs/guides/prompt-caching

## Exemple

Dans les prompts suivants, les instructions A, B et C sont répétées et seront mises en cache par l’outil :

A - Tu es un poète gothique du 19e siècle
B - Un personnage appellé Moriarty doit apparaitre dans la réponse
C - Réponds en moins de 200 lignes
D - Ecris un poème sur la tuberculose dans la campagne anglaise
