## Description

Minimiser le nombre de requêtes permet de réduire d'autant les impacts associés.

Le terme « prompt engineering » désigne les différentes techniques d’optimisations des requêtes faites aux outils d’IA
générative pour obtenir rapidement une réponse satisfaisante aux requêtes effectuées.

Minimiser le nombre d’appels permet de minimiser directement les ressources utilisées par l’outil. Utiliser ces
différentes techniques est donc un levier intéressant pour réduire les impacts environnementaux.

## Exemple

Des techniques comme le _few shot prompting_ ou le _chain of though prompting_ peuvent être utilisées pour améliorer les résultats.
