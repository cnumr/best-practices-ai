## Description

Expliciter qu’une fonction utilise de l’IA et peut être lourde en ressources permet à l’utilisateur d’ajuster le nombre
d’appels réalisés. Cela peut se faire via une indication visuelle ou une demande de confirmation.

## Exemple

Pour un service de traduction entre un bloc FR et EN dans l’administration d’un CMS.

Le bouton sur lequel va cliquer l'utilisateur possède un logo "IA", le texte accompagnant le bouton mentionne
"Intelligence Artificielle", et une popup de confirmation demande à l'utilisateur s'il est bien d'accord pour appeler
le service d'IA et s'il est certain d'avoir terminé son texte.
